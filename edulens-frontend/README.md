# EduLens AI Learning Intelligence Platform - Frontend

This is a fully isolated frontend-only project for the EduLens AI Learning Intelligence Platform. It serves as a visual representation of the EduLens product experience while remaining frontend-only and easy to run locally, making it perfect for a GitHub portfolio, a hackathon demo UI, or an interactive frontend prototype.

## Features

- **Modern AI SaaS Dashboard Design:** Professional enterprise-grade UI using glassmorphism and soft UI depth.
- **Dark / Light Mode:** Fully responsive theme toggle system.
- **Single Page Application (SPA):** Custom vanilla JS hash-based router for seamless navigation between views.
- **Dynamic Content & Mock Data:** Simulates real-time AI insights, analytics, and behavior logs using `mock-data.json`.
- **Pure Canvas / SVG Charts:** Dynamic charts built from scratch with JS—no external heavy libraries.

## Setup Instructions

This project has **zero backend dependencies** and **zero build steps** (no Node.js required).

1. Clone or download the repository.
2. Open the `edulens-frontend` folder in VS Code.
3. Use the **Live Server** extension to serve the `index.html` file, or simply drag and drop `index.html` into a modern web browser.
*(Note: Using Live Server is recommended to ensure `fetch()` requests for `mock-data.json` work correctly due to CORS policies on local files).*

## Tech Stack

- **HTML5:** Semantic structure.
- **CSS3:** Vanilla CSS with custom properties (variables), Flexbox, Grid, and animations. No Tailwind or Bootstrap.
- **JavaScript (ES6+):** Pure Vanilla JS for state management, component rendering, routing, and interactivity. No React, Vue, Angular, or jQuery.

## Purpose

This project is built to demonstrate a premium UI/UX design and interaction flow for an AI-powered educational system. It illustrates how high-quality user interfaces can be constructed efficiently with pure web standards.
